Here is the description of the problem:

泊松分酒问题
有3个无刻度的容器，容量分别为A升，B升，C升,	A>B>C。其中A中装满酒，另外两个空着。
要求你只用这三个容器操作，最后使得某个容器中正好有D升酒。
Task	1：对A=12， B=8， C=5,	D=6,	打印出最简单的一种方案。
Task	2:	对于一般情况，判断是否有解，如果有解，给出最简单的一种方案。
